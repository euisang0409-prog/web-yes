# 운영체제 정리 (시험 대비)

## 1. 운영체제(OS)

* 하드웨어와 소프트웨어를 사용자가 편리하게 사용할 수 있도록 돕는 시스템 소프트웨어
* 자원(CPU, 메모리 등)을 효율적으로 배분하는 총괄 관리자

### 선택 기준

* 서버 개발: Linux
* 일반 PC: Windows
* 애플 앱 개발: MacOS

---

## 2. 권한

### 관리자 권한

* 시스템 전체 제어 가능
* 설정 변경 가능
* 프로그램 설치 가능

### 일반 권한

* 개인 작업만 가능
* 시스템 변경 제한

---

## 3. 메모리 관리

### 메모리 누수

* 사용이 끝난 메모리를 반환하지 않는 상태
* 프로그램이 계속 메모리를 잡고 있어 오류 발생

### GC (Garbage Collector)

* Java에서 사용하지 않는 객체를 자동으로 회수

---

## 4. CPU, 코어, 스레드

* CPU: 한 번에 하나의 작업 실행 (빠르게 번갈아 처리)
* 코어: 실제 연산 장치
* 스레드: 프로그램 실행 흐름

---

## 5. 로그(Log)

* 프로그램 실행 기록
* 문제 해결의 핵심 자료

---

## 6. 운영체제 주요 기능

1. 처리 능력 향상
2. 응답 시간 단축
3. 신뢰성 향상
4. 인터페이스 제공
5. 입출력 관리
6. 프로그램 및 사용자 보호
7. 메모리 관리
8. CPU 관리
9. 파일 관리
10. 명령어 해석 및 실행

---

## 7. 오픈소스

* 소스코드가 공개된 소프트웨어

### 장점

1. 비용 절감
2. 개발 속도 향상
3. 검증된 코드
4. 커뮤니티 지원
