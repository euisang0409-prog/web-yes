# 운영체제 / 리눅스 / 사용자 / 권한 통합 정리

---

## 1. 운영체제(OS)

- 하드웨어와 소프트웨어 사이를 관리하는 프로그램
- 종류: Windows, MacOS, iOS, Android, Linux, Unix

### 운영체제 선택 기준

- 웹 개발: Windows
- 서버 운영: Linux, Unix
- 앱 개발: MacOS
- 앱 실행: iOS, Android

---

## 2. 운영체제가 하는 일

- CPU 관리
- 주기억장치(RAM) 관리
- 파일 관리 (생성, 삭제, 수정)
- 입출력 장치 관리
- 사용자 인터페이스 제공
- 프로그램 실행
- 보안 및 보호
- 안정적인 시스템 유지

---

## 3. 서술형 핵심 문제

### Q1. 여러 프로그램 동시에 실행

기능:

- CPU 실행 관리
- 주기억장치 관리

이유:

- CPU를 나눠 사용해야 함
- 메모리를 적절히 분배해야 함

---

### Q2. 메모리 부족 오류

기능:

- 주기억장치 관리

이유:

- 메모리가 부족하면 프로그램 실행 불가

---

### Q3. 서버 느려짐

확인 항목:

- CPU 사용량
- RAM 사용량

## 4. Windows 명령어

```bash
dir          # 목록
cd 경로      # 이동
copy A B     # 복사
move A 경로  # 이동
del 파일     # 삭제
mkdir 폴더   # 폴더 생성
type 파일    # 내용 출력
cls          # 화면 지우기
ipconfig     # IP 확인
ping 주소     # 네트워크 테스트
```

---

## 5. Linux 명령어

```bash
ls           # 목록
ls -l        # 상세
ls -a        # 숨김 포함
cd 경로      # 이동
cp           # 복사
mv           # 이동/이름변경
rm           # 삭제
rmdir        # 폴더 삭제
cat 파일     # 내용 출력
pwd          # 현재 위치
clear        # 화면 지우기
touch 파일   # 파일 생성
echo "text"  # 출력
```

---

## 6. touch 명령어

- 파일 생성 또는 시간 변경

```bash
touch 파일
touch -c 파일
touch -t 202604211030 파일
touch -d "2026-04-21 10:30:00" 파일
```

시간 종류:

- atime: 접근 시간
- mtime: 수정 시간
- ctime: 상태 변경 시간

---

## 7. echo 활용

```bash
echo "World" >> file.txt    # 추가
```

````
## 13. 사용자 관리

```bash
sudo adduser 아이디
sudo passwd 아이디
id 아이디
sudo userdel 아이디
sudo userdel -r 아이디
````

---

## 14. 그룹 관리

```bash
sudo groupadd 그룹명
sudo groupdel 그룹명
sudo usermod -aG 그룹 아이디
sudo gpasswd -d 아이디 그룹
```

---

## 15. 관리자 권한

```bash
sudo usermod -aG sudo 아이디
```

---

## 16. 파일 권한

권한:

- r (4)
- w (2)
- x (1)

예:

```bash
rw-r--r--
```

숫자:

```bash
chmod 644 파일
```

---

## 17. chmod

```bash
chmod u+x 파일
chmod g-w 파일
chmod o-r 파일
```

---

## 18. chown

```bash
sudo chown 사용자 파일
sudo chown 사용자:그룹 파일
sudo chown -R 사용자:그룹 디렉터리
```
