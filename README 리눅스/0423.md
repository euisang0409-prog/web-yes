# 1. 프로토콜 (Protocol)

- 컴퓨터끼리 통신할 때 지켜야 하는 규칙

## 핵심 구성요소
- 구문(Syntax): 데이터 형식
- 의미(Semantics): 데이터 의미 및 동작  
  - HTTP 요청 메서드: GET, POST, PUT, DELETE  
  - 응답 코드: 2XX, 3XX, 4XX, 5XX  
- 타이밍(Timing): 전송 순서, 속도, 타이밍

## 대표 프로토콜
- HTTP / HTTPS: 웹 통신
- FTP: 파일 전송
- TCP: 정확하고 안정적인 통신
- UDP: 빠른 통신

# 2. OSI 7계층

## 1계층 물리계층
- 데이터: Bit
- 전기/빛/전파 신호
- 장비: 허브, 리피터

## 2계층 데이터링크 계층
- 데이터: Frame
- MAC 주소 사용
- 장비: 스위치, 브리지

## 3계층 네트워크 계층
- 데이터: Packet
- IP 주소 사용
- 장비: 라우터, L3 스위치

## 4계층 전송 계층
- 데이터: Segment
- TCP/UDP
- 장비: L4 스위치, 로드밸런서

## 5계층 세션 계층
- 연결 생성, 유지, 종료

## 6계층 표현 계층
- 데이터 변환, 암호화

## 7계층 응용 계층
- 사용자와 직접 연결
- 웹, 메일, 파일 전송
