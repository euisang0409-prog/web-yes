# 문제 39

현재 상황:

센서 데이터 발생량:

- 초당 20건 예상

유사 프로젝트:

- 초당 50건 예상
- 실제 초당 150건 발생

동일한 증가율을 적용한다고 할 때 최종 예상 처리량을 계산하고, 왜 재조정이 필요한지 작성하시오.

## 정답

최종 예상 처리량:

- 초당 60건

계산:

50 → 150 = 3배 증가

20 × 3 = 60

재조정 이유:

- 초기 예상값만 사용하면 실제 사용량 증가 시 성능 저하나 시스템 장애가 발생할 수 있다.
