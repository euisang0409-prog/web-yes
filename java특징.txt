종류:
정수형
byte	      1byte
short 	      2byte
int 	      4byte
long 	      8byte

변수를 만드는 방법

1. 변수의 선언

자료형 변수명; -> 값이 들어있지 않은 상태

2. 변수에 값을 대입

변수명 = 값;

대입은 원하는 만큼 가능하다.
대입을 하면 이전의 값은 사라진다.

3. 변수의 초기화(initialize)

자료형 변수명 = 값;


참조자려형

참조자료형 -> 값이 있는 주소값을 알고 있다.
