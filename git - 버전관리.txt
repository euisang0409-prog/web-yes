git -> 버전관리 

init(initialize) : 초기화 -> 초기값을 지정한다.

저장소 (repository)
버전저장 (commit)
원격 저장소로 업로드 (push)
원격 저장조에서 다운로드 (pull) 

내가 작업한 내용을 스테이지에 올린다.
add 명령어를 사용한다.


git add  파일이름 파일이름 파일이름 ->  해당 파일만 추가
git add .  ->  현재 폴더의 모든 변경점을 추가

버전 기록 : git commit -m "메시지"

어떤 버전이 기록되어 있는지 확인 : git log

되돌리기 : git reset 옵션 커밋ID

옵션 : --SOFT -> 커밋 내용은 사라지는데 내용을 임시 저장 상태로 되돌린다.

--MIXED -> 임시 저장도 취소한다. 작업공간에는 남겨둠

-- HARD -> 임시 저장도 취소, 작업공간에서도 삭제

HEAD : 최신 버전을 의마한다. 

HEAD ~2 : 두번 전 으로 

원격저장소에 올리기 전 반드시 커밋이 완료된 상태여야 한다.

 add . 	commit	push
폴더 -> stage -> 버전기록 -> 원격 저장소에 저장
