## 1. Stream 최종 연산 특징

// 1. 스트림을 종료시키는 연산

// 3. 결과를 반환하거나 소비한다
// 4. 한 번 사용된 스트림은 재사용 불가

---

## 2. count() - 개수 세기

```java id="c1k8p2"
import java.util.*;

public class Main {
    public static void main(String[] args) {

        List<String> words = Arrays.asList("java","spring","db","react");

        long count = words.stream()
                .filter(s -> s.length() >= 5)
                .count();

        System.out.println("길이가 5이상인 요소 개수: " + count);
    }
}
```

---

## 3. anyMatch() - 하나라도 만족하면 true

```java id="a2k9p1"
List<Integer> numbers = Arrays.asList(1,3,5,8);

boolean result = numbers.stream()
        .anyMatch(x -> x % 2 == 0);

System.out.println(result);
```

---

## 4. allMatch() - 모두 만족해야 true

```java id="b3m8q2"
result = numbers.stream()
        .allMatch(x -> x % 2 == 0);

System.out.println(result);
```

---

## 5. noneMatch() - 하나도 만족하지 않으면 true

```java id="d9n3p7"
result = numbers.stream()
        .noneMatch(x -> x % 2 == 0);

System.out.println(result);
```

## 7. reduce() - 누적 연산

```java id="r8k2m1"
int sum = numbers.stream()
        .reduce(0, (a, b) -> a + b);

```

## 9. 기본형 스트림 최종 연산

```java id="i8m3q1"
import java.util.stream.*;
```

---

### sum()

```java id="s1k8p9"
int sum = IntStream.rangeClosed(1, 5).sum();
System.out.println(sum);
```

---

### average()

```java id="a8m2p7"
OptionalDouble avg = IntStream.of(10, 20, 30).average();

// 기본값 처리
double result = avg.orElse(0.0);

// 값이 있을 때만 실행
avg.ifPresent(System.out::println);
```

---

### max()

```java id="m7k2p1"
OptionalInt max = IntStream.of(7, 3, 2, 9).max();
max.ifPresent(System.out::println);
```

---

### min()

```java id="n2k8q5"
OptionalInt min = IntStream.of(7, 3, 2, 9).min();
min.ifPresent(System.out::println);
```
