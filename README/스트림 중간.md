# Java Stream 중간 연산 정리 (복구본)

---

## 1. Stream 중간 연산 특징

```java
// 1. 중간 연산은 Stream을 반환한다
// 2. 단독으로 결과를 만들 수 없다 (최종 연산 필요)
// 3. 메서드 체이닝으로 연결 가능
// 4. 최종 연산이 있어야 실행된다 (lazy evaluation)
```

---

## 2. filter() - 조건 필터링

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        List<Integer> list = Arrays.asList(1,2,3,4,5,6);

        // 짝수만 출력
        list.stream()
            .filter(x -> x % 2 == 0)
            .forEach(System.out::println);

        // 원본은 변경되지 않음
        System.out.println(list);
    }
}
```

---

## 3. map() - 요소 변환

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        List<String> sList = List.of("java","spring","react");

        // 대문자 변환
        sList.stream()
            .map(String::toUpperCase)
            .forEach(System.out::println);
    }
}
```

## 4. mapToInt() - 기본형 변환

import java.util.*;

        List<String> list2 = List.of("a","bb","ccc");

## 5. sorted() - 정렬

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        int[] arr = {5,2,9,1,7};

        // 오름차순
        Arrays.stream(arr)
            .sorted()
            .forEach(System.out::println);

        System.out.println();

        // 내림차순
        Arrays.stream(arr)
            .boxed()
            .sorted((a,b) -> b - a)
            .forEach(x -> System.out.println(x + " "));
    }
}
```

## 7. distinct() - 중복 제거

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        int[] iArr = {1,1,2,3,3,3,4,4,5,6};

        Arrays.stream(iArr)
            .distinct()
            .forEach(x -> System.out.print(x + " "));
    }
}
```

## 8. limit() - 개수 제한

```java
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {

        IntStream.rangeClosed(1, 10)
            .limit(3)
            .forEach(x -> System.out.print(x + " "));
    }
}
```

## 9. skip() - 건너뛰기

## 10. peek() - 중간 확인 (디버깅용)

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        List<String> sList = List.of("java","spring","react","db");

        sList.stream()
            .filter(s -> s.length() >= 5)
            .peek(s -> System.out.print("필터 통과:" + s))
            .map(String::toUpperCase)
            .peek(s -> System.out.print("변환 후:" + s))
            .forEach(x -> System.out.print(x + " "));
    }
}
```

## 11. Stream 핵심 정리

```java
// filter   : 조건 필터링
// map      : 데이터 변환

// distinct : 중복 제거
// limit    : 개수 제한
// skip     : 건너뛰기
// peek     : 중간 확인
```
