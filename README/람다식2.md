# Java 컬렉션 + 람다 핵심 정리

---

# 1. forEach()

컬렉션 요소를 하나씩 꺼내서 처리

list.forEach(s -> System.out.println(s));

list.forEach(System.out::println);

---

# 2. removeIf()

조건이 true인 요소 삭제

nums.removeIf(x -> x % 2 == 0); // 짝수 삭제

names.removeIf(n -> n.contains("민")); // "민" 포함 삭제

---

# 3. replaceAll()

List 전체 요소 변경

names2.replaceAll(name -> name.toUpperCase());

---

# 4. sort()

정렬 기준 지정

names2.sort((a, b) -> a.compareTo(b));

---

# 5. compareTo()

문자열 사전순 비교

a.compareTo(b);

- 음수: a가 앞
- 0: 같음
- 양수: a가 뒤

---

# 6. Map.forEach()

key, value 동시에 출력

map.forEach((key, value) -> System.out.println(key + " " + value));

---

# 7. Map.replaceAll()

모든 value 변경

map.replaceAll((key, value) -> value + 500);

---

# 💡 핵심 요약

forEach → 출력  
removeIf → 삭제  
replaceAll → 변경  
sort → 정렬  
Map.forEach → key/value 출력
