# Windows 명령어 정리

## 1. 파일/폴더 관리 명령어

mkdir 폴더명 : 폴더 생성
cd 경로 : 폴더 이동

---

## 2. 파일 목록 확인

dir : 현재 폴더의 파일 목록 출력

---

## 3. 파일 삭제

del 파일명

---

## 4. 폴더 삭제

rmdir 폴더명 /s /q
/s : 내부 파일까지 삭제
/q : 확인 없이 삭제

---

## 5. IP 주소 확인

ipconfig

---

## 6. 네트워크 연결 확인

ping 주소 : 인터넷 연결 테스트

---

## 7. 실행중인 프로세스 확인

tasklist

---

## 8. 현재 사용자 확인

whoami

---

## 9. 출력 명령어

echo 내용 : 입력한 내용을 그대로 출력

---

## 10. 파일 생성 (echo 사용)

echo 내용 > 파일명 : 파일 생성 및 덮어쓰기
예: echo . > test.java

> : 덮어쓰기
>
> > : 추가 쓰기

---

## 11. 파일 내용 출력 (type)

type 파일명 : 파일 내용을 터미널에 출력

- 로그 확인
- 설정 파일 확인
- 텍스트 파일 내용 확인

### 여러 파일 출력

type 파일명1 파일명2 : 여러 파일을 이어서 출력

### 파일 합치기

type 파일명1 > 파일명2 : 파일1 내용을 파일2로 저장
type 파일명1 파일명2 > 파일명3 : 두 파일을 합쳐 파일3 생성

### 키보드 입력

type con : 키보드 입력을 화면에 출력

- 종료: Ctrl + Z 후 Enter

type con > 파일명 : 키보드 입력을 파일로 저장

---

# 과제

바탕화면에 myproject 폴더를 만들고 안에 src, bin 폴더를 생성하시오

---

# 정답 명령어

cd %USERPROFILE%\Desktop
mkdir myproject
cd myproject
mkdir src
mkdir bin

---

# 한 줄 실행 버전

cd %USERPROFILE%\Desktop && mkdir myproject && cd myproject && mkdir src bin
