# Java Stream 실전 예제 (filter + map + forEach)

---

## 1. 숫자 리스트 - 짝수만 + 10 더하기

```java
import java.util.*;

public class Example {
    public static void main(String[] args) {

        List<Integer> numbers =
                Arrays.asList(3,5,7,8,10,4,1,2,6,9,1,1,1,3,3,3);

        // 짝수만 골라서 10을 더한 뒤 출력
        numbers.stream()
                .filter(x -> x % 2 == 0)   // 짝수 필터
                .map(x -> x + 10)          // +10 변환
                .forEach(System.out::print);

        System.out.println();
    }
}
```

---

## 2. 문자열 리스트 - 길이 조건 + 대문자 변환

```java
import java.util.*;

public class Example {
    public static void main(String[] args) {

        List<String> words =
                Arrays.asList("java", "spring", "react", "db", "server");

        // 길이 5 이상만 필터링 후 대문자 변환
        words.stream()
                .filter(x -> x.length() >= 5)   // 길이 조건
                .map(String::toUpperCase)       // 대문자 변환
                .forEach(x -> System.out.println(x + " "));

        System.out.println();
    }
}
```

// filter → 조건 걸기
