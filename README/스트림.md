# Java Stream 전체 정리

---

## 1. 스트림 생성

// 컬렉션에서 생성

```java
List<String> list = Arrays.asList("A","B","C");
Stream<String> stream = list.stream();

Set<Integer> set = new HashSet<>(Arrays.asList(1,2,3,4,5));
Stream<Integer> setStream = set.stream();
```

// Map에서 생성 (stream() 없음)

```java
Map<String, Integer> map = new HashMap<>();
map.put("사과",1000);
map.put("바나나",2000);

// key만
map.keySet().stream();

// value만
map.values().stream();

// key + value
map.entrySet().stream();
```

// 배열에서 생성

```java
String[] fruits = {"사과","바나나","포도"};

Arrays.stream(fruits);
Stream.of(fruits);
```

// 기본형 스트림

```java
int[] arr = {1,2,3};

IntStream stream = Arrays.stream(arr);
IntStream.of(1,2,3);

IntStream.range(1, 5);       // 1 ~ 4
IntStream.rangeClosed(1, 5); // 1 ~ 5

IntStream.iterate(0, n -> n + 1);

new Random().ints(5);
```

## 2. 중간 연산 (Intermediate Operation)

// filter: 조건에 맞는 요소만 추출

```java
list.stream()
    .filter(s -> s.equals("A"))
    .forEach(System.out::println);
```

// map: 데이터 변환

```java
list.stream()
    .map(s -> s.toLowerCase())
    .forEach(System.out::println);
```

// sorted: 정렬

```java
list.stream()
    .sorted()
    .forEach(System.out::println);
```

// distinct: 중복 제거

```java
Stream.of("A","A","B","C")
      .distinct()
      .forEach(System.out::println);
```

// limit: 개수 제한

```java
IntStream.range(1, 10)
         .limit(5)
         .forEach(System.out::println);
```

---

## 3. 최종 연산 (Terminal Operation)

// forEach: 반복 출력

```java
list.stream().forEach(System.out::println);
```

// count: 개수

```java
long count = list.stream().count();
```

// sum (IntStream)

```java
int sum = IntStream.of(1,2,3).sum();
```

// max / min

```java
OptionalInt max = IntStream.of(1,2,3).max();
OptionalInt min = IntStream.of(1,2,3).min();
```

// findFirst

```java
Optional<String> first = list.stream().findFirst();
```

## 4. collect (중요)

// 리스트로 변환

```java
List<String> result =
    list.stream()
        .filter(s -> s.equals("A"))
        .collect(Collectors.toList());
```

// Set으로 변환

```java
Set<String> result =
    list.stream()
        .collect(Collectors.toSet());
```

// 문자열 합치기

```java
String result =
    list.stream()
        .collect(Collectors.joining(","));
```

## 5. Optional 처리

// 값 존재 여부 확인

```java
Optional<String> opt = list.stream().findFirst();

if(opt.isPresent()) {
    System.out.println(opt.get());
}
```

// 기본값 설정

```java
String result = opt.orElse("없음");
```
