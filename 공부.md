for
- 반복횟수가 정해져 있을 때 사용하는 반복문
```
for (초기식; 조건; 증감식) {
    // 반복 실행할 코드
}
```

이중 for문
- for 문 안에 for문이 있는 형태
- 바깥쪽 for문이 한바퀴 돌 때 안쪽 for문은 반복을 전부 수행한다. 

예시:
for (초기식1; 조건식1; 증감식1) {
	for ( 초기식2; 조건식2; 증감식;) {
		실행문
	}
}

기타제어문
break
- 반복문 내부에서 break를 만나게 되면 반복문 그 즉시 종료

contiue
- 반복문 내부에서 continue를 만나게 되면 증감식이나 조건식으로 돌아가게 된다.

중첩 반복문일 때 기타제어문은 가장 가까이 있는 반복문에 대해
적용이 되기 때문에 바깥쪽 반복문을 빠져나가려면 라벨을 달아야 한다.
예시 :
 for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;				실행 결과: 1 2 4 5 --> 3은 건너뜀
    }							i = 3 → continue → 출력 안 하고 다음으로
    System.out.println(i);
}							
continue 만나면 아래 코드 실행 안 함
