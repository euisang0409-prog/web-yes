정규표현식(Regular Expression)
문자열의 "패턴"을 정의하는 규칙
특정 문자열이 내가 원하는 형식으로 적혀있는지 검사

a-z : 소문자
A-Z : 대문자
0-9 : 0부터 9까지
가-하 : 모든 한글
? : 없거나 한개
+ : 한개거나 이상
[] : 안에 적힌것중 한개 ex) [abc] -> a,b,c중 한글자

문자열.matches(정규식) : 1회성 검사
