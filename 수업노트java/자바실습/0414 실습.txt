2. ProductService 클래스 작성

ArrayList<Product>를 사용하여 상품 목록을 관리하시오.

다음 기능을 구현하시오.

1) 상품 등록
public void addProduct(Product product)
상품을 등록한다.
같은 code가 이미 존재하면 "이미 존재하는 상품 코드입니다."를 출력한다.

2) 전체 상품 조회
public void printAllProducts()

3) 상품 코드로 조회
public Product findByCode(String code)

4) 상품 정보 수정
public void updateProduct(String code, Function<Product, Product> updater)
상품을 찾아서 updater를 적용한다.
없으면 "상품이 존재하지 않습니다."를 출력한다.

5) 조건에 맞는 상품 조회
public void findProductsByCondition(Predicate<Product> predicate)

6) 조건에 맞는 상품 삭제
public void deleteProductsByCondition(Predicate<Product> predicate)

3. Main 클래스에서 다음 작업 수행

다음 상품들을 등록하시오.

new Product("P001", "삼각김밥", 1200, 10)
new Product("P002", "콜라", 2000, 0)
new Product("P003", "도시락", 4500, 5)

그리고 아래 작업을 수행하시오.

실행 작업
전체 상품 출력
Function<Product, Product>를 이용하여 P001 가격을 10% 인상
Function<Product, Product>를 이용하여 P003 상품명의 앞뒤 공백 제거
Predicate<Product>를 이용하여 재고가 있는 상품만 조회
Predicate<Product>를 이용하여 품절 상품(stock == 0) 삭제
삭제 후 전체 상품 출력

출력 예시
=== 전체 상품 목록 ===
Product{code='P001', name='삼각김밥', price=1200, stock=10}
Product{code='P002', name='콜라', price=2000, stock=0}
Product{code='P003', name='도시락', price=4500, stock=5}

=== 재고가 있는 상품 ===
Product{code='P001', name='삼각김밥', price=1320, stock=10}
Product{code='P003', name='도시락', price=4500, stock=5}

=== 품절 상품 삭제 후 ===
Product{code='P001', name='삼각김밥', price=1320, stock=10}
Product{code='P003', name='도시락', price=4500, stock=5}
