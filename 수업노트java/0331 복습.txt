생성자 : 객체를 메모리에 올리는 역할, 필드를 초기화 하는 역할

메서드 : 만들고 싶은 기능, 자주사용하는 기능을 만드는 것

this : 호출한 객체 자신
this.필드명
this()

supre : 부모클래스
super.필드명
super.()
super.메서드명()

오버라이딩(메서드 재정의)

상속관계에서만 사용할 수 있음

부모클래스에 있는 메서드 이름으로 사용하고 싶지만
자식클래스의 상황에 맞지 않아서 자식클래스에서 재정의할 때

접근제한자, 반환형, 메서드명, 매개변수는 그대로 사용

내용만 수정

추상화
중요한 공통점만 뽑아내고, 불필요한 세부분은 감추는 것

추상클래스
공통된 특징은 가지고 있지만, 그 자체로 완전한 객체가 되기 어려운 클래스
- 공통적인 필드
- 공통적인 메서드

자식클래스들이 추상클래스를 상속해서 구체적인 내용을 작성
