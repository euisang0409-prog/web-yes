Object클래스(모든 클래스의 최상위 클래스)

equals() : 주소가 같은지 비교

hashcode() : 주소를 반환

toString() : 화면에 출력을 어떻게 할 것인지

대부분의 클래스들이 오버라이딩하여 위 메서드들을 사용
==========================================
롬복 설치
1. 이클립스가 설치된 곳에 롬복파일 넣기
2. 롬복파일 실행하기 
3. 이클립스 실행파일 설정하고 설치하기
4. 이클립스가 켜져있다면 껏다켜기
5. 라이브러리파일을 프로젝트 내부에
서도 등록하기
