상속
ㄴ 관련있는것들끼리 묶는 것
공통적인 속성(필드, 멤버)이나 동작(메서드)을 부모쪽에 몰아놓는것

자식은 부모클래스를 상속받아서 사용
is-a 관계
강아지는 동물이다.(dog is  a animal)
N차 상속이 가능함 상속 -> 상속 -> 상속
꼭 부모가 아닌 조상의 필드나 메서드를 사용할 수 있다.
Car -> Bus -> SchoolBus
다중상속이 불가능하다.
Object : 모든 클래스의 부모클래스

Arrays.toString(배열)

sealed : 봉인된 클래스
permit : 내가 허용한 클래스만 상속을 할 수 있게 한다.

public sealed class Person permits  


인터페이스
일반 필드 X
상수, 추상메서드, 디폴트메서드, 정적메서드, private 메서드
ㄴ 동작에 대한 가이드라인 제공
같은 범주에 있는 것들이 아니어도됨
fly();
class Bird{};
class Ariplane{};

추상클래스와 마찬가지로 직접 객체를 만드는 것이 불가능하다.
인터페이스를 구현할 클래스가 있어야 한다.

class Impl Implements 인터페이스명 {};

추상메서드를 반드시 오버라이딩해야 한다.

다중구현이 가능하다.
class Impl implements inteface1, interface2{};
- 각 인터페이스에 있던 추상메서드를 전부 오버라이딩 해야한다.

다중 상속이 가능하다.
- 상속은 인터페이스 끼리만 가능하다.
interface interA extends InterB, InterC {};
