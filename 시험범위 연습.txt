# 05/13 SQL 시험 범위 총정리

## 1. DDL

### CREATE TABLE

```sql
CREATE TABLE student(
    student_id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
);
```

### 제약조건

```sql
PRIMARY KEY
FOREIGN KEY
NOT NULL
UNIQUE
DEFAULT
CHECK
```

### FOREIGN KEY 예제

```sql
CREATE TABLE department(
    department_id INT PRIMARY KEY,
    department_name VARCHAR(50)
);

CREATE TABLE employee(
    employee_id INT PRIMARY KEY,
    name VARCHAR(50),
    department_id INT,
    FOREIGN KEY(department_id)
    REFERENCES department(department_id)
);
```

---

# 2. DML

## SELECT

```sql
SELECT *
FROM student;
```

### WHERE

```sql
SELECT *
FROM student
WHERE age >= 20;
```

### ORDER BY

```sql
SELECT *
FROM student
ORDER BY age DESC;
```

### LIKE

```sql
SELECT *
FROM student
WHERE name LIKE '%김%';
```

### BETWEEN

```sql
SELECT *
FROM product
WHERE price BETWEEN 10000 AND 50000;
```

### IN

```sql
SELECT *
FROM product
WHERE category IN ('전자기기','가구');
```

---

## INSERT

```sql
INSERT INTO student(student_id,name,age)
VALUES(1,'홍길동',20);
```

---

## UPDATE

```sql
UPDATE student
SET age = 25
WHERE student_id = 1;
```

---

## DELETE

```sql
DELETE FROM student
WHERE student_id = 1;
```

---

# 3. 집계함수

## COUNT

```sql
SELECT COUNT(*)
FROM student;
```

## SUM

```sql
SELECT SUM(price)
FROM product;
```

## AVG

```sql
SELECT AVG(price)
FROM product;
```

## MAX / MIN

```sql
SELECT MAX(price), MIN(price)
FROM product;
```

---

# 4. GROUP BY

```sql
SELECT category, COUNT(*)
FROM product
GROUP BY category;
```

### 부서별 평균 급여

```sql
SELECT department_id, AVG(salary)
FROM employee
GROUP BY department_id;
```

---

# 5. HAVING

- 그룹 결과 조건

```sql
SELECT category, COUNT(*)
FROM product
GROUP BY category
HAVING COUNT(*) >= 2;
```

---

# 6. INDEX

## 생성

```sql
CREATE INDEX idx_product_name
ON product(product_name);
```

## 특징

- 조회 속도 향상
- WHERE 검색 최적화
- 너무 많으면 INSERT/UPDATE 성능 저하 가능

---

# 7. SUBQUERY

## 단일행 서브쿼리

```sql
SELECT *
FROM employee
WHERE salary >
(
    SELECT AVG(salary)
    FROM employee
);
```

## IN 서브쿼리

```sql
SELECT *
FROM employee
WHERE department_id IN
(
    SELECT department_id
    FROM department
    WHERE location = '서울'
);
```

---

# 8. JOIN

## INNER JOIN

```sql
SELECT e.name, d.department_name
FROM employee e
INNER JOIN department d
ON e.department_id = d.department_id;
```

## LEFT JOIN

```sql
SELECT e.name, d.department_name
FROM employee e
LEFT JOIN department d
ON e.department_id = d.department_id;
```

---

# 9. VIEW

## 생성

```sql
CREATE VIEW high_salary_employee AS
SELECT *
FROM employee
WHERE salary >= 5000;
```

## 조회

```sql
SELECT *
FROM high_salary_employee;
```

## 삭제

```sql
DROP VIEW high_salary_employee;
```
