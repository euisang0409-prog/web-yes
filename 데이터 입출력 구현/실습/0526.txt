실습 시나리오

- A 학원은 현재 수강생 정보를 엑셀로 관리하고 있다.
- 출결은 강사가 별도 엑셀 파일에 입력하고, 상담 기록은 상담사가 개인 PC 문서 파일에 저장한다.
- 결제 내역은 관리자가 은행 입금 내역을 확인하여 수기로 관리한다.
- 새로운 학원 통합 관리 시스템에서는 다음 업무를 처리하려고 한다.

업무   설명
수강생 등록   수강생의 이름, 연락처, 이메일, 수강 과정을 등록한다.
출결 입력   강사가 수강생의 출석일, 입실 시간, 퇴실 시간, 출결 상태를 입력한다.
상담 등록   상담사가 수강생의 상담일, 상담 내용을 등록한다.
결제 등록   관리자가 수강생의 결제일, 결제 금액, 결제 상태를 등록한다.
과정 관리   관리자가 과정명, 개강일, 종강일, 담당 강사를 등록한다.

- 시스템은 향후 문자 발송 시스템과 연계하여 수강생에게 공지 문자를 발송할 예정이다.
- 문자 발송에는 수강생 이름, 연락처, 메시지 내용이 필요하다.
- 출결 상태는 반드시 다음 네 가지 값 중 하나만 사용한다.

```
출석, 지각, 조퇴, 결석
```

- 결제 상태는 반드시 다음 세 가지 값 중 하나만 사용한다.

```
결제완료, 미납, 환불
```

사용자 권한은 다음과 같이 구분한다.
사용자   권한
관리자   모든 데이터 조회, 등록, 수정 가능
강사   담당 과정의 수강생 정보 조회, 출결 등록 가능
상담사   수강생 정보 조회, 상담 등록 가능
수강생   본인 정보와 본인 출결만 조회 가능
-------------------------------------------------------------------------------
- 문제1
    - 위 시나리오에서 **수강생 등록 업무**에 사용되는 입력데이터 3개를 작성하시오.
    
    ### 조건
    
    - **수강생 등록 업무**에 제시된 데이터만 사용한다.
    - 출결, 상담, 결제, 과정 관리 데이터는 작성하지 않는다.
    - 정확히 3개만 작성한다.

정답 : 이름, 연락처, 이메일

- 문제2
    - 위 시나리오에서 문자 발송 시스템과 연계되는 데이터를 모두 작성하시오.
    
    ### 조건
    
    - 문자 발송 시스템에 필요하다고 명시된 데이터만 작성한다.

정답 : 이름, 연락처 메시지 내용


- 문제3
    - 위 시나리오에서 필요한 엔터티 5개를 작성하시오.
    
    ### 조건
    
    - 업무 단위가 아니라 데이터로 관리해야 하는 대상을 작성한다.
    - 시나리오의 주요 업무명에서 도출한다.
    - 정확히 5개를 작성한다.

정답:
1. 수강생
2. 출결
3. 상담
4. 결제
5. 과정

- 문제4
    - 출결 상태에 해당하는 공통코드 값을 모두 작성하시오.
    
    ### 조건
    
    - 시나리오에 제시된 출결 상태 값만 작성한다.
    - 값을 추가하거나 바꾸지 않는다.

정답 : 출석일, 입실 시간, 퇴실 시간, 출결 상태

- 문제5
    - 결제 상태에 해당하는 공통코드 값을 모두 작성하시오.

    ### 조건
    
    - 시나리오에 제시된 결제 상태 값만 작성한다.
  정답 :  결제일, 결제 금액, 결제 상태

- 문제6
    - 수강생 엔터티의 속성 4개를 작성하시오.
    
    ### 조건
    
    - 수강생 등록 업무에 제시된 데이터만 사용한다.

정답: 수강생의 이름, 연락처, 이메일, 수강 과정

- 문제7
    - 출결 엔터티의 속성 4개를 작성하시오.
    
    ### 조건
    
    - 출결 입력 업무에 제시된 데이터만 사용한다.

정답 : 수강생의 출석일, 입실 시간, 퇴실 시간, 출결 상태

- 문제8
    - 수강생과 출결의 관계를 작성하시오.
    
    ### 조건
    
    - 한 명의 수강생이 여러 날짜의 출결 기록을 가질 수 있다는 기준으로 판단한다.
    - 관계는 1:1, 1:N, N:M 중 하나로 작성한다.

정답 : 1:N

- 문제9

    ### 조건
    
    - 출결 입력은 새로운 출결 기록을 저장하는 업무이다.
    - C, R, U, D 중 하나로 작성한다.

정답 : C

- 문제10
    - 강사가 접근할 수 있는 데이터와 수행할 수 있는 작업을 작성하시오.
    
    ### 조건
    
    - 시나리오의 사용자 권한 표를 기준으로 작성한다.
    - 강사 권한만 작성한다.
    - 관리자, 상담사, 수강생 권한은 작성하지 않는다.

정답 : 담 출결 입력

    | attendance |

    | attendance_id |

- 문제12
    - 다음과 같이 관계가 설계되었다.
        
        ```
        수강생 : 출결 = 1 : 1
        ```
        
    - 하지만 실제 업무에서는 한 명의 수강생이 여러 날짜의 출결 기록을 가질 수 있다.
    - 위 내용을 기준으로 **수강생과 출결 사이의 올바른 관계 유형을 작성하시오.**
    
    **조건**
    
    - `1:1`, `1:N`, `N:M` 중 하나만 작성한다.
    - 설명은 작성하지 않는다.

정답 : 1:N
