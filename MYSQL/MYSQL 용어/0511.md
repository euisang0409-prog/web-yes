# SQL 반복문 정리

## WHILE 문

- 조건이 TRUE인 동안 반복 실행
- 조건을 먼저 검사한다.

```sql
WHILE 조건 DO
    실행문;
END WHILE;
```

### 예제

```sql
DELIMITER //

CREATE PROCEDURE while_example()
BEGIN

    DECLARE v_num INT DEFAULT 1;

    WHILE v_num <= 5 DO

        SELECT v_num;

        SET v_num = v_num + 1;

    END WHILE;

END //

DELIMITER ;
```

---

## REPEAT 문

- 무조건 한 번 실행 후 조건 검사
- UNTIL 조건이 TRUE이면 종료

```sql
REPEAT
    실행문;
UNTIL 조건
END REPEAT;
```

### 예제

```sql
DELIMITER //

CREATE PROCEDURE repeat_example()
BEGIN

    DECLARE v_num INT DEFAULT 1;

    REPEAT

        SELECT v_num;

        SET v_num = v_num + 1;

    UNTIL v_num > 5

    END REPEAT;

END //

DELIMITER ;
```

---

## LOOP 문

- 무한 반복 기반
- 직접 종료 조건을 만들어야 한다.
- LEAVE를 사용하여 반복 종료

```sql
label: LOOP

    실행문;

    IF 조건 THEN
        LEAVE label;
    END IF;

END LOOP;
```

### 예제

```sql
DELIMITER //

CREATE PROCEDURE loop_example()
BEGIN

    DECLARE v_num INT DEFAULT 1;

    my_loop: LOOP

        SELECT v_num;

        SET v_num = v_num + 1;

        IF v_num > 5 THEN
            LEAVE my_loop;
        END IF;

    END LOOP;

END //

DELIMITER ;
```

---

## 반복문 비교

| 반복문 | 특징 |
|---|---|
| WHILE | 조건 먼저 검사 |
| REPEAT | 무조건 1번 실행 후 검사 |
| LOOP | 무한 반복 기반, 직접 종료 |

---

## 핵심 정리

- WHILE : 조건 기반 반복
- REPEAT : 최소 1회 실행
- LOOP : LEAVE로 종료
