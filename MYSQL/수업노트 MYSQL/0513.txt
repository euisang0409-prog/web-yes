Stroded Program
- MySQL에서 프로그래밍적으로 동작하는 프로그램
- 프로시저 -> 여러개의 sql을 한 번에 처리할 수 있는 함수
- funcion -> 값을 반환하는 함수
- handler -> 예외처리
- cursor -> 한 행씩 데이터를 꺼내서 처리하는 문법
- trigger  -> insert, update, delete 문이 실행됐을 때 이벤트의 처리

JAVA 와 MYSQL 연동하기

java가 MySQL과 연결이 되려면 JDBC Driver가 필요하다.
jdbc(java database connectivity) driver
- java 프로그램과 데이터베이스를 연결해주는 통로 역할을 한다.
- java와 DB를 연결
- SQL을 전달하는 역할
- 결과를 받아오는 역할

DBMS 회사마다 JDBC Driver가 다르다
MySQL : mysql-commerctor-j
Oracle : ojdbc
PostgreSQL : postgresql
