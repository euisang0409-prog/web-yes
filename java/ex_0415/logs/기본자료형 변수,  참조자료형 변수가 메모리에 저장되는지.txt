기본자료형 변수 (Primitine Type)
종류: int , double , char , boolean 등 
특징: 값 자체를 변수에 저장
메모리 위치: 스택(Stack)
변수 선언 시 바로 값이 저장됨	
예를 들어:	int num = 10;
		
		int num = 10;
	num 이라는 변수에는 10이라는 값 자체가 저장됨
	
	변수명-> num		값-> 10

2. 참조자료형 변수 (Reference Type)
종류: String , 배열, 객체 등
특징: 실제 데이터는 힙(Heap)에 저장, 변수에는 참조 주소 저장
메모리 위치:
변수: 스택(Stack)
객체/배열 데이터: 힙(Heap) 

/********************************************
서술형 답안

기본자료형 변수와 참조자료형 변수는 메모리에 저장되는 방식에서 차이가 있다.
기본자료형 변수는 int, double, char, boolean과 같은 자료형으로, 변수 자체에 실제 데이터 값이 직접 저장된다. 이 값은 주로 스택(Stack) 메모리 영역에 저장되며, 변수명을 통해 해당 값을 바로 사용할 수 있다.

예 : int a = 10; 			a -> 10 a안에 10이 들어와있음

/********************************************

반면 참조자료형 변수는 String, 배열, 클래스 객체 등과 같은 자료형으로, 변수에는 실제 데이터가 아니라 데이터가 저장된 메모리의 주소값이 저장된다. 실제 데이터는 힙(Heap) 메모리 영역에 생성되며, 변수는 이 주소를 참조하여 데이터에 접근한다.

따라서 기본자료형은 값을 직접 저장하는 방식이고, 참조자료형은 주소를 통해 데이터를 간접적으로 참조하는 방식이라는 차이가 있다. 또한 참조자료형도 출력 시에는 실제 값이 보이지만, 이는 저장된 주소를 통해 해당 데이터를 찾아 출력하기 때문이다.

예 : String s = "hello";		s = 주소 	"hello" = 힙에 저장
