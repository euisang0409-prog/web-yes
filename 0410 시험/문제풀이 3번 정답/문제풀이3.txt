[문항 설명] 

다음은 쇼핑몰의 '회원 등급별 할인 금액 계산기'를 만들기 위한 프로그램 설계서와 미완성 코드입니다. 

설계서의 조건을 확인하여 코드를 완성하고, 오류를 수정하시오.

[프로그램 설계서]

변수: 원가(정수형 price)와 회원 등급(문자열 grade)을 입력받는다.
조건: 
* grade가 "VIP"인 경우: 20% 할인
grade가 "GOLD"인 경우: 10% 할인
그 외의 경우: 할인 없음 (0%)
 
출력: 최종 할인된 금액(discountedPrice)을 계산하여 출력한다.
 

public class DiscountCalculator {
   public static void main(String[] args) {
       int price = 10000;
       String grade = "VIP";
       int discountedPrice = 0;
 
       // [A] 오류발생로직
       if (grade == "VIP") { 
           discountedPrice = price * 0.8; 
       } 

       // [B] 나머지 로직

   }
}

1-2. 설계서의 모든 조건과 [A]에서 찾은 오류를 반영하여, 올바르게 동작하는 전체 Java 코드를 서술하시오.
