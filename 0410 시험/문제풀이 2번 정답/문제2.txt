1. 프로그램명
StudentScoreManager
 
2. 프로그램 목적
학생 1명의 이름과 3과목 점수를 입력받아
총점, 평균, 합격 여부를 출력하는 프로그램을 작성한다.
 
3. 입력 항목
학생 이름(String)
Java 점수(int)
DB 점수(int)
HTML 점수(int)
 
4. 처리 조건
총점을 계산한다.
평균은 실수형으로 처리한다.
평균이 60점 이상이면 "합격"
평균이 60점 미만이면 "불합격"으로 출력한다.
각 점수는 0 이상 100 이하라고 가정한다
 

출력 형식 예시

이름: 홍길동
총점: 240
평균: 80.0
결과: 합격
